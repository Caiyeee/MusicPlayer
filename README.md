### MusicPlayer
##### 主功能：播放音乐  伴奏录音
#####
##### 歌曲：手机里的音乐文件
##### 录音：所有用户的录音文件
##### 伴奏：文件名后缀为“-instru”的音乐文件
##### 本地录音：当前用户的录音文件
#####
##### 录音结束后自动命名并保存，命名方式：原歌名 + “-record” + 用户id，例如“songName-record1”
